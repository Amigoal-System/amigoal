# **App Name**: Amigoal MVP

## Core Features:

- Social Authentication: User authentication via Google, Apple, Facebook, or email.
- User Profiles: Display user profiles with individual statistics and values.
- Training Module: Input, evaluate, and display training data. Currently no token rewards.
- Match Module: Game preparation and analysis to display results
- Tasks Module: Tasks management tool to manage deadlines and status
- Document Module: Allow upload of documents
- Members Module: Club and team member management. Not linked to the wallet/tokens features for the MVP.
- PWA Support: Progressive Web App (PWA) readiness for installability and offline functionality.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to evoke trust and reliability, reflecting the platform's role in managing sports data. Use shadcn's color palette for variations.
- Background color: Light blue (#E8F0FE), a desaturated shade of the primary, creates a clean and professional backdrop.  Use shadcn's color palette for variations.
- Accent color: Analogous cyan (#34A853) to provide visual interest without clashing, drawing the eye to key actions.  Use shadcn's color palette for variations.
- Body and headline font: 'PT Sans' (sans-serif) to create a balance between a modern look and a touch of personality. Adhere to shadcn's typography scale.
- Use clear, minimalist icons related to sports and management, ensuring easy recognition. Use shadcn's icon library.
- Design a clean and intuitive layout that allows users to easily navigate between modules and manage their data. Adopt shadcn's grid system.
- Subtle transitions and loading animations to enhance user experience. Base animations on animate.js
